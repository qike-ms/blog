# blog

Qi's blogposts. Drafted in obsidian, promoted here when ready to publish.

## Layout

```
posts/
  YYYY-MM-DD-slug.md    # one file per post
drafts/                 # work-in-progress (optional)
```

## Workflow

1. Draft in obsidian.
2. When ready, copy to `posts/` with YAML frontmatter:
   ```yaml
   ---
   title: ...
   date: YYYY-MM-DD
   tags: [...]
   ---
   ```
3. Commit + push to `main`.
4. Optionally cross-post to LinkedIn / Substack / X.
